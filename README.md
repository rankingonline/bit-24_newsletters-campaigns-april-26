# Bitrix24 Newsletters - Abril 2026

Este repositorio contiene una colección de 5 plantillas de correo electrónico (newsletters) diseñadas para **Bitrix24**, enfocadas en las novedades, automatización y adopción de CRM para abril de 2026.

## 🚀 Vista Previa

Para ver todas las plantillas de forma interactiva, abre el archivo principal en tu navegador:
- [**index.html**](./index.html) - Portal central con diseño Glassmorphism para navegar entre newsletters.

## 🛠️ Detalles Técnicos de Maquetación

Las plantillas han sido desarrolladas siguiendo los estándares más estrictos de **Email Marketing**:

- **Compatibilidad 100%**: Maquetación basada en tablas (`<table>`) para asegurar que se vean correctamente en Outlook (Desktop/Web), Gmail, Apple Mail y otros clientes.
- **Responsive Design**: Ajuste automático para dispositivos móviles mediante media queries (`max-width: 600px`).
- **Inline CSS**: Todos los estilos críticos están inyectados directamente en las etiquetas para evitar bloqueos de estilo.
- **Tipografía Premium**: Uso de la fuente **Inter** (Google Fonts) con fallbacks seguros.
- **Optimización de Imágenes**: Estructura preparada para imágenes compuestas que mantienen la estética del diseño original.
- **Pesos de Fuente**: Se aplicó un standard de `font-weight: 600` para los elementos destacados.
